# Yelp clone built with ReactJS

This is a repo with code for [this tutorial](https://www.fullstackreact.com/articles/react-tutorial-cloning-yelp/)

## Setup
	git clone https://github.com/rickydunlop/yelp-clone yelp
	cd yelp
	npm install
	
## Running locally

	npm start
	
## Testing 
	
	npm test
